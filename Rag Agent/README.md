# Company Knowledge Base — RAG Agent

Retrieval-augmented chat agent for the company's FAQ / services / pricing docs, with a web chat UI, per-session conversation history, and calendar/lead integrations. 

Answers are generated by **Google Gemini 3.1 Flash-Lite** (using the new `llama-index-llms-google-genai` integration).

## Features

- **Google Calendar Integration**: The agent checks calendar availability and books meetings seamlessly.
- **Google Sheets Lead Capture**: Stores lead information including name, email, meeting date, and **home address** in the tracking spreadsheet.
- **Gmail Confirmation Emails**: Sends a confirmation email to the client automatically upon successful booking.
- **Ultra Token-Efficient**:
  - **Pruned Memory**: Keeps only the last 2 turns (4 messages) of conversation history to save tokens.
  - **ReAct Step Filtering**: Automatically strips out tool outputs, system notes, and thoughts from the history database.
  - **2-Line Concise Replies**: Every response is limited to a maximum of 2 lines. If more details are available, it summarizes them and asks if you want to see them.
- **Free Local Embeddings**: Uses `BAAI/bge-small-en-v1.5` (~130MB, runs locally on CPU).

## Setup

1. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Ingest Documents**:
   ```bash
   python ingest.py
   ```
   This processes the markdown files inside the `data/` directory and creates a persistent Chroma vector database in `chroma_db/`.

3. **Configure Environment**:
   Copy `.env.example` to `.env` and fill in the required keys:
   ```bash
   # Add your Google Gemini API Key and Sheets configuration
   GEMINI_API_KEY=your_api_key
   SPREADSHEET_ID=your_google_sheet_id
   ```

4. **Authenticate Google Services**:
   Place your `credentials.json` (OAuth Desktop Application credentials from Google Cloud Console) in the project directory, then run the interactive authentication script:
   ```bash
   python auth.py
   ```
   This will authenticate Google Sheets, Calendar, and Gmail, and save the authorization token as `token.json`.

## Run the Web UI

```bash
python server.py
```
Open http://localhost:8000 to interact with the web chat interface.

## Run the Agent from the CLI

```bash
python agent.py                                          # interactive loop
python agent.py "does tier 2 include 3D renderings?"      # one-shot
```

## Test Retrieval Only (No LLM/API Key Needed)

```bash
python query.py "does tier 2 include 3D renderings?"
```

## Project Structure

- `agent.py` — The core agent definition. Initializes the `GoogleGenAI` model and custom ReAct workflow with tools, and implements context-aware session history pruning.
- `google_services.py` — Google Calendar, Sheets, and Gmail integrations.
- `chunker.py` — Markdown-aware document chunker for structured ingestion.
- `ingest.py` — Processes and indexes data to ChromaDB.
- `auth.py` — Handles Google OAuth interactive verification.
- `server.py` — FastAPI server backing the Web UI.
- `templates/index.html` — Responsive glassmorphic web UI.
